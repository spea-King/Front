# 실전 면접 연습 서비스 Frontend UI 명세서

## 전체 화면 흐름

```
[Landing]
   ↓[기업/직무 선택]
   ↓[이력서·자소서 업로드]
   ↓[면접 세션 설정]
   ↓[실전 면접 진행]
   ↓[면접 종료]
   ↓[사후 리포트]
```

## 1. Landing 페이지

**목적**: 서비스 컨셉 즉시 이해
**디자인 방향**: "프리미엄 프로페셔널" - 딥 네이비 + 골드 액센트

### 와이어프레임
```
┌─────────────────────────────┐
│   실전 면접 연습 서비스     │
│                             │
│  "답을 알려주는 AI가 아니라 │
│   말하는 연습을 돕습니다"   │
│                             │
│[ 면접 연습 시작 ]           │
└─────────────────────────────┘
```

### 요구사항
- 요소 최소화 (제목, 설명, CTA 버튼만)
- CTA 버튼: "[ 면접 연습 시작 ]" - 호버 시 골드 그라데이션
- 배경: 미묘한 그라데이션 메쉬 (#0a1628 → #1a2744)
- 애니메이션: 페이지 로드 시 텍스트 staggered reveal
- 폰트: Playfair Display (제목) + Source Sans Pro (본문)

## 3. 이력서/자소서 업로드 페이지

**목적**: 지원자 정보 입력
**디자인 방향**: "클린 폼" - 넉넉한 여백

### 와이어프레임
```
┌─────────────────────────────┐
│  지원자 정보 입력           │
│                             │
│[ PDF 업로드 영역 ]          │
│    이력서 / 자소서          │
│                             │
│  (내용은 저장되지 않습니다) │
│                             │
│[ 다음 ]                     │
└─────────────────────────────┘
```

### 기능
- PDF 드래그앤드롭 또는 클릭 업로드
- 파일명 표시 + 삭제 버튼
- 지원 형식: PDF만
- 파일 크기 제한: 10MB
- 보안 문구: "(내용은 저장되지 않습니다)"
- 업로드 완료 시 다음 버튼 활성화
- 에러 상태: 빨간색 테두리 + 에러 메시지

## 4. 면접 세션 설정 페이지

**목적**: 면접 조건 설정
**디자인 방향**: "컨트롤 패널" - 세그먼트 컨트롤

### 와이어프레임
```
┌─────────────────────────────┐
│  면접 설정                  │
│                             │
│  면접 질문 수 [5개 ▼]       │
│                             │
│  면접관 목소리 [남][여]     │
│                             │
│  면접 스타일                │
│  [친절한] [압박]            │
│                             │
│[ 면접 시작 ]                │
└─────────────────────────────┘
```

### 기능
- 질문 수: 5개~10개 슬라이더/드롭다운 (기본값: 5개)
- 면접관 목소리: 토글 (남 / 여)
- 면접 스타일: 세그먼트 (친절한 / 압박)
- 설정 미리보기 표시
- "면접 시작" 버튼 클릭 시 면접관 사진과 함께 시작

## 5. 실전 면접 진행 페이지 (핵심)

**목적**: 실제 면접 진행
**디자인 방향**: "몰입형 인터뷰"

### 와이어프레임
```
┌─────────────────────────────┐
│[면접관 사진 (pulse 애니메이션)]│
│                             │
│  🔊 "자기소개 부탁드립니다"   │
│                             │
│  ────────────────────────   │
│  ⏱️ 01:45 / 02:00          │
│  ████████████████░░░░░░░    │
│                             │
│  📌 답변 구조 가이드 [▼]    │
│  • 도입                     │
│  • 핵심 경험                │
│  • 결과 / 배운 점           │
│                             │
│[ 답변 시작 ] [ 종료 ]       │
└─────────────────────────────┘
```

### 기능
1. **면접관 음성 재생**: 질문 텍스트 + 음성
2. **타이머**: 2분 (120초) 카운트다운
   - 속도에 따른 색상 변화: green → yellow → red
   - 2분 경과 시 자동 다음 질문
3. **답변 가이드**: 접을 수 있는 패널
   - 도입 → 핵심 경험 → 결과/배운 점
4. **버튼**: "답변 시작" / "종료"
5. **질문 순서**:
   - Q1: 자기소개 (항상 첫 질문)
   - Q2~Q6: Frontend Developer 포커스 포인트 기반

### 상태 관리
```typescript
interface InterviewState {
  currentQuestionIndex: number;
  remainingTime: number;
  isRecording: boolean;
  answers: string[];
}
```

## 6. 면접 종료 화면

**목적**: 분석 중 연출
**디자인 방향**: "로딩 서스펜스" - 긴장감/기대감

### 와이어프레임
```
┌─────────────────────────────┐
│  면접이 종료되었습니다      │
│                             │
│     [분석 애니메이션]        │
│                             │
│      분석 중...             │
│  답변 구조 분석 중...        │
│                             │
│  ████████████░░░░ 85%       │
│                             │
│      [ 결과 보기 ]          │
│         (fade in)           │
└─────────────────────────────┘
```

### 기능
- 분석 애니메이션: 돋보기/차트/파형 움직임
- 변경되는 서브텍스트:
  - "답변 구조 분석 중..."
  - "언어 패턴 분석 중..."
  - "피드백 생성 중..."
- 진행률: 0% → 100% (3-5초)
- 완료 후 "[ 결과 보기 ]" 버튼 fade in
- 배경: 딥 네이비 + subtle 파티클/웨이브

- 말 속도 (텍스트 수 / 답변 시간으로 계산)
- 각 질문별 시간 막대그래프

### 기능

#### 섹션 1: 전체 요약
- 3줄 요약 (긍정적/개선점)
- 평균 답변 시간 (ex: 55초)
- 말 속도 (텍스트 수 / 시간)
- 시간 분포 막대그래프
- 면책 조항: "*합격 여부와 무관한 연습용 피드백입니다"

#### 섹션 2: 질문별 상세 분석
각 질문:
- 질문 제목 + 답변 시간 + 말 속도
- 막대그래프 (시간 시각화)
- "내 답변" (텍스트)
- "AI 답변 제안" (더 나은 답변 예시)
- "한 줄 평가" (강점/약점 요약)

## 기술 스택

- **Framework**: React + TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Charts**: Recharts
- **Icons**: Lucide React
- **State**: Zustand / Context API
- **Upload**: react-dropzone

## 공통 디자인 시스템

### 색상 팔레트
- **Primary**: 딥 네이비 (#0a1628, #1a2744)
- **Accent**: 골드 (#d4af37, #c9a227)
- **Success**: 초록 (#22c55e)
- **Warning**: 주황 (#f97316)
- **Error**: 빨강 (#ef4444)
- **Background**: 화이트/네이비 그라데이션

### 타이포그래피
- **제목**: Playfair Display
- **본문**: Source Sans Pro
- **코드**: JetBrains Mono

### 애니메이션
- **페이지 전환**: fade + slide
- **hover**: subtle lift (translateY -2px)
- **로딩**: pulse, shimmer
- **타이머**: 색상 변화 + 프로그레스
